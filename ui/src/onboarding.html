vim-anywhere

Vim keybindings for every text field on macOS

Accessibility

Required to read and modify text in any application.

Input Monitoring

Required to intercept keyboard events globally.

How It Works

  • Press Esc to enter Normal mode in any text field
  • Use Vim motions: hjkl, w, b, ciw, etc.
  • Press i to go back to typing
  • Terminals and non-text elements are automatically excluded
